# Marketing Templates

Reusable sales page and bonus page templates for affiliate marketing campaigns.

## Templates Included

### 1. Sales Page Template (`sales-page-template.html`)
A high-converting, modern sales page with:
- Hero section with 3-part headline structure
- Problem agitation section
- Features grid
- Offer section with pricing
- Guarantee box
- Mobile responsive design

### 2. Bonus Page Template (`bonus-page-template.html`)
A professional bonus delivery page with:
- Success confirmation banner
- Bonus details and value proposition
- Download section
- Support contact information
- Mobile responsive design

## How to Use

### Step 1: Customize Variables
Each template uses `{{VARIABLE_NAME}}` placeholders. Replace these with your content:

#### Sales Page Variables:
- `{{PRODUCT_NAME}}` - Your product name
- `{{HEADLINE}}` - Page title
- `{{PRIMARY_COLOR}}` - Main brand color (e.g., #3b82f6)
- `{{SECONDARY_COLOR}}` - Secondary color (e.g., #1d4ed8)
- `{{ACCENT_COLOR}}` - CTA button color (e.g., #f59e0b)
- `{{TEXT_COLOR}}` - Body text color (e.g., #1f2937)
- `{{BG_COLOR}}` - Background color (e.g., #ffffff)
- `{{FONT_HEADING}}` - Heading font (e.g., 'Inter')
- `{{FONT_BODY}}` - Body font (e.g., 'Inter')
- `{{PATTERN_INTERRUPT}}` - Bold hero headline
- `{{MECHANISM_REVEAL}}` - How it works
- `{{OUTCOME_IDENTITY}}` - Transformation promise
- `{{CTA_TEXT}}` - Button text
- `{{PROBLEM_TITLE}}` - Problem section heading
- `{{PROBLEM_PARAGRAPH_1-3}}` - Problem description
- `{{FEATURES_TITLE}}` - Features section heading
- `{{FEATURE_1-3_TITLE}}` - Feature names
- `{{FEATURE_1-3_DESC}}` - Feature descriptions
- `{{OFFER_TITLE}}` - Offer section heading
- `{{OFFER_SUBTITLE}}` - Offer description
- `{{PRICE}}` - Product price (numbers only)
- `{{CHECKOUT_URL}}` - Payment link
- `{{GUARANTEE_TITLE}}` - Guarantee heading
- `{{GUARANTEE_TEXT}}` - Guarantee details
- `{{YEAR}}` - Current year
- `{{COMPANY_NAME}}` - Your company name

#### Bonus Page Variables:
- `{{BONUS_NAME}}` - Bonus name
- `{{BONUS_VALUE}}` - Bonus value (numbers only)
- `{{BONUS_DESCRIPTION}}` - Bonus details
- `{{FEATURE_1-4}}` - Bonus features
- `{{DOWNLOAD_URL}}` - File download link
- `{{DOWNLOAD_BUTTON_TEXT}}` - Button text
- `{{DOWNLOAD_NOTE}}` - Download instructions
- `{{SUCCESS_MESSAGE}}` - Confirmation message
- `{{HEADER_TITLE}}` - Page header
- `{{HEADER_SUBTITLE}}` - Header subtitle
- `{{SUPPORT_TEXT}}` - Support message
- `{{SUPPORT_EMAIL}}` - Support email

### Step 2: Deploy to Vercel

1. Fork or clone this repository
2. Connect to Vercel
3. Deploy instantly

### Step 3: Reuse for New Campaigns

1. Duplicate the template file
2. Replace all `{{VARIABLES}}`
3. Deploy as new project
